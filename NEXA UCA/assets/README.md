# Assets

Esta carpeta contiene todos los recursos visuales utilizados dentro de la aplicación
movil NEXA UCA.

Los assets permiten mantener organizada la identidad visual del proyecto y centralizar 
elementos gráficos reutilizables dentro de la interfaz

## Estructura 

### Icons/
Contiene los íconos utilizados dentro de la aplicación 

Ejemplos:

-  Íconos de navegación
-  Íconos de productividad
-  Íconos de configuración 
-  Íconos de notificaciones 
------

###  Images/ 

Contiene imágenes generales utilizadas en distintas pantallas de la aplicación.

Ejemplos:

- Imágenes decorotivas
- Fondos 
- Recursos visuales informativos

---
### logos/

Contiene los logos oficiales relacionados con la identidad visual del proyecto.

Ejemplos:

- Logo principal NEXA UCA 
- Variantes del logo
- Versiones transparentes 
- Versiones monocromáticas 

---
### Illustrations/

Contiene ilustraciones utilizadas para enriquecer visualmente la experiencia del usuario.

Ejemplos:

- Ilustraciones onboarding 
- Estados vacios
- Recursos gráficos UI

---
### Animations/

Contiene animaciones utilizadas dentro de la aplicación 

Ejemplo:

- Animaciones Lottie 
- Transiciones visuales
- Indicadores animados
